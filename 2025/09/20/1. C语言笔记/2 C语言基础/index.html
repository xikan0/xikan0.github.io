1.3数据的运算

数制基本概念

指用一组固定的符号和统一的规则来表示数值的方法，按进位的原则进行计数的方法，称为进位计数制

十六进制： 0 ~ 9 到 A ~ F（不区分大小写）

十进制：2D 八进制：2O 十六进制：2H

数制转换 （重点）

十进制转二、八、十六进制

除2（8、16）取余，倒序输出

小数部分：乘（2,8,16）取整顺序输出

例：78.6875D = ___B

原码、反码、补码

  1. 正数的原码、反码、补码相同
  2. 负数的反码=原码除符号位取反
  3. 负数的补码=负数的反码+1

注意：

  1. 计算机在内存中存储方式是补码
  2. 全体补码形似环，超过“最大数”即为“最小数”，反之同理
  3. 一个正数因递增变负数：向上发生“溢出”
  4. 一个负数因递减变正数：向下发生“溢出”

1.4数据类型

特点：

  1. 基本类型的数据有原子性，不可分解
  2. 不能定义 void 型变量
  3. 指针类型标识存储对象的地址
  4. 构造类型的数据可分解成若干“元素”，每个元素可以是基本类型也可以是其他构造类型

1.4.1整形数据

1）基本整形（int型）

占2个或4个字节（具体编译系统指定），在存储单元中存储的方式是：整数的补码形式存放，一个正数的补码是此数的二进制形式

当为负数时，其补码是二进制取反，再加一

最左一位是用来表示符号的，0表示正数，1表示负数

当2字节时，整形分配变量范围是$2^{15}-1$到$-2^{15}$次方，即32767~ -32768

当4字节时，整形分配变量范围是$-2^{31}$到$2^{31}-1$，即-2 147 483 648 ~ 2 147 483 647

2）整形变量的符号属性

2.运算符和表达式

结合性：

  1. 左结合：自左向右[a + b + c → (a + b) + c]
  2. 右结合：自右向左[a = b = c → a = (b = c)]

表达式的运算顺序规则

  1. 从左向右扫描字符，逐个分析运算符和操作数
  2. 括号最优先
  3. 优先级较高先运算
  4. 优先级相同，则按运算符的结合方向处理

优先级最高的运算符先与操作数进行绑定（按照结合性），如果相同，则按结合性顺序绑定

各种运算符冲类：

算数运算符：加减乘除、取余、++ --

条件运算符：?:（三元运算符）

逗号运算符：,

  • 多数C编译系统采用**“向零取整”**方法，即$5-3.2=1$,$-5+3.2=-1$，取整后向零靠拢
  • %运算要用整数
  • 除%以外运算符操作数都可以是任何算数类型

  1. 自增自减运算符不能用于常量和表达式，如：--(x+y);++8;都是非法的
  2. **后自增自减(a++)**是自左向右的结合方式，**前自增自减(++a)**是自右向左的结合方式，后自增自减优先级大于前自增自减
  3. 在&&和||之间，优先级仅决定表达式的结合方式，而求值顺序由短路特性控制

执行路径

一、运算符优先级决定执行顺序

优先级高的运算符会先于优先级低的运算符进行计算

二、逻辑运算符的短路求值特性影响路径

在逻辑与&&和逻辑或||运算中，如果能够根据前面的操作数确定整个表达式的结果，则不会计算后面的操作数，这称为短路求值

三、逗号运算符

在执行路径中，会按照从左到右的顺序依次计算每个表达式，但整个逗号表达式的值是最后一个表达式的值

C语言中，除法运算符'/'，是重载的运算符，定义了两种不同的运算：

  1. 浮点除法：两个浮点数相除，商是浮点数
  2. 整数除法：两个整数相除，商是整数

浮点数和整数相除，整数自动转换为浮点数

浮点数据不能参与求余运算（%）

负数参与求余运算，则是对两数的绝对值求余，符号与被除数相同

2.2不同类型混合运算

  1. 加减乘除运算中有float或double型，结果是 double 型
  2. 如果 int 型与 float 或 double 型进行运算，其结果是 double 型
  3. 字符(char)与整形数据进行运算，就是ASCII代码与整形进行计算
  4. 字符(char)与实型数据进行运算，则将ASCII代码转换为 double 型数据

例：10+'a'+i*f-d/3

  1. 进行10+'a'运算，因为'a'是97，则结果 107
  2. 进行i*f运算，先将 i 和 f 都转换成 float 型，运算结果为 7.5
  3. 后 107+7.5 先将 107 转换成单精度数，相加结果为 114.5，float 型
  4. 进行 d/3 运算，先将3转换成 double 型，结果为 2.5，double 型
  5. 将 10+'a'+i*f 的结果 114.5 与 d/3 的结果 2.5 相减， 结果为 112.0，double 型

2.3sizeof运算符

sizeof(变量名或类型)：获取类型或变量的类型的大小

易错点：

  1. sizeof 可直接空格加变量名，获得该变量的类型的大小
  2. sizeof 并非函数，是在编译时获取类型大小，而函数是在执行时获得函数值的
  3. sizeof int 是错误方法，sizeof 用于类型关键字必须加括号
  4. sizeof 只获取类型大小，括号内如果有表达式也不进行计算，如++a无效

3.C语句

3.1 C语句的作用和分类

一个函数由执行部分和声明部分组成

执行部分：由语句组成，向计算机系统发出操作指令，要求执行相应的操作，一个C语句经过编译后产生若干条机器指令

声明部分：对有关数据的声明

C语言分为：控制语句、函数调用语句、表达式语句、空语句、复合语句

语句是一个表达式的最后加上一个分号构成

控制语句

C语言只有九种控制语句：

  1. if()…else… 条件语句
  2. for()… 循环语句
  3. while()… 循环语句
  4. do…while() 循环语句
  5. continue() 终止本次循环语句
  6. break 终止执行switch或循环语句
  7. switch 多分支选择语句
  8. return 从函数返回语句
  9. goto 转向语句

函数调用语句

一个函数调用加一个分号构成

表达式语句

一个表达式加一个分号构成，典型的是赋值语句，也就是赋值表达式

一个表达式加分号就成了一个语句

空语句

空语句里只有一个分号

复合语句

用{}将多个语句和声明括起来成为复合语句，又称语句块

赋值语句

赋值运算符=

将一个数据或表达式的值赋给一个变量

复合的赋值运算符

易错点：x+=y%3——>x=x+(y%3)

赋值过程中的类型转换

如果赋值运算符两侧的类型一致，则直接进行赋值

如果两侧类型不一致，但都是基本类型时，进行类型转换（自动进行的）

  1. 将浮点型赋给整形时，对浮点数取整，再赋予整型变量

  2. 将整形数据赋值给浮点型变量时，数值不变，但以浮点形式存储

  3. 将一个double类型数据赋给float 变量时，先将双精度数转换为单精度，即只取6~7位有效数字，远超过float范围时会显示INF

    将一个float类型数据赋给double变量时，数值不变，有效数字扩展到15位

  4. 字符型变量赋值给整型变量时，将字符的ASCII代码赋给整型变量

  5. 将占一个字节多的整形数据赋给一个占字节少的整型变量时（如int赋给short或char），会直接截断，只保留低字节，多出来的消失

z (C99)修饰整数格式符，指定输出一个size_t整数，这个类型由sizeof返回

一、1.234500e+004=1.234500*($10^4$)=12345

二、8.000000e-001=8*($10^{-1}$)=8*0.1

*使用举例：

总的来说，*号可以替代格式修饰符中使用的数字，当个占位符，但必须在后面的变量列表里，按顺序给出具体的值或变量值

注意：

  1. 可以指定域宽，即在%和格式字符中间加上列数%5d
  2. 通常在有格式字符o和x按八进制和十六进制形式输出整数时，在数值前不出现0和0x，可在%和格式字符间插入#来实现。
    例：printf（“%o，%#o，%x，%#x\n”，10，10，10，10）；
    输出：12，012，a，0xa
  3. 指定输出宽度的同时指定左对齐可在宽度前插入“-”来实现。

4.2 scanf函数

scanf函数的格式字符和printf函数基本相同

注意：

  1. scanf函数中的格式控制后面应当是变量地址，而不是变量名
  2. 格式控制字符串中的普通字符都要打出来
  3. **再用%c格式声明时，空格字符和转义字符中的字符都能作为有效字符输入 **
  4. 在输入数据时，如输入空格、回车、Tab或非法字符（不属于数值的字符），认为该数据结束

输入格式修饰符

修饰符 功能
* 抑制分配，或者说跳过该输入项
digit(s) 最大域宽，在达到最大域宽或遇到空白符时停止输入
hh 读取一个char或unsigned char
如%hhd、%hhu
ll(L小写) 读取一个long long或unsigned long long
如%lld、%llu
h l L %hd和%hi表示读取一个short int
%ho、%hx和%hu表示读取一个unsigned short int
%ld和%li表示读取一个long
%lo、%lx和%lu表示读取一个unsigned long
%le、%lf和%lg表示读取一个double
%Le、%Lf和%Lg表示读取一个long double
如果没有使用这些修饰符%d %i %o %x表示int，%e、%f和%g表示float
j (C99)当修饰一个整数格式符时表示使用intmax_t或uintmax_t类型
z (C99)当修饰一个整数格式符时表示使用sizeof的返回类型
t (C99)当修饰一个整数格式符时表示使用ptrdiff_t类型

scanf停止的标志：

  1. 遇空格符、回车符、制表符
  2. 达到输入的域宽
  3. 遇到不能解析的字符

注意：

  1. 连续输入多个整数或浮点数时，数据之间用一个或多个空格、换行符、制表符分隔
  2. 输入字符%c时，空格和换行符也会算进去（读取字符串%s时不会）
  3. 指定了域宽后，输入多个整数时，不必隔开
  4. scanf函数每次开始读取，会跳过所有的空白字符，直到遇到第一个非空白字符的时候才开始读取(读取除字符除外)

上述代码中，当我键入”12345”回车后，首先”12345\n”会进入输入缓冲区，然后scanf读入”12345”，回车”\n”被留在了缓冲区

如果下面调用了getchar()函数，那回车就会被读入，通常要避免这种情况，这时我们可以在scanf读取完12345后立刻调用getchar函数将 ‘ \n ‘读走

scanf空格注意点

scanf在双引号里碰到空格时，会触发一个机制，会开始顺序扫描输入缓冲区，跳过所有空白字符（回车，空格，制表等），直到扫描到非空白字符，才会继续输入

上述代码中，两个%c格式声明之间有一个空格，这时，由于scanf每次读取都会跳过所有的空白字符（回车，空格，制表），所以当输入第一个字符后，如果一直输入空白字符，则程序不会停止，直到输入第二个非空白字符为止

4.3 字符输入输出函数

putchar函数

putchar(c)：输出c字符，如果c是整数（0~127），则输出其ASCII码对应的字符

其中c可以是字符常量、整形常量、字符变量或整型变量（值在ASCII码范围内）

getchar函数

getchar的值是int型

在用键盘输入时，不是敲一个字符就立即送到计算机中，而是存在键盘的缓冲器里，按了Enter键后才送入计算机中，如果输入了多个字符，那他就只读一个，其他的在缓冲区里，如果程序由调用getchar了，就从里面再拿一个

回车键和空格也是可以作为一个字符的，可以被这两个函数读取

4.4 字符串输入输出函数

gets函数

char* gets(char*s)：从键盘输入，以回车结束的字符串放入数组中，并自动加'\0'

  1. 输入串长度应小于字符数组维数
  2. 与scanf不同，gets函数并不以空格作为字符串输入结束的标志

puts函数

int puts(const char*s)：向显示器输出字符串（输出完后自动换行）

字符数组必须以'\0'结束，否则可能会导致未定义行为
