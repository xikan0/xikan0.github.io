C语言支持逻辑判断和运算，但没有逻辑类型

0为假，非0为真，任何类型数据都可以作逻辑值

  3. if语句内也可以内嵌if语句

关系运算符和关系表达式

关系运算符优先级

优先级相同（高）：< <= > >=

优先级相同（低）：== !=

算数运算符 > 关系运算符 > 赋值运算符

关系表达式：用关系运算符将两个数值或数值表达式连接起来的式子，关系表达式的值是逻辑值，“真”或“假”

逻辑运算符和逻辑表达式

&&：逻辑与 ||：逻辑或 !：逻辑非

逻辑与和逻辑或是双目运算符，逻辑非是单目运算符

优先级：

非 > 与 > 或

！> 算数运算符 > 关系运算符 > &&和 || > 赋值运算符

逻辑表达式：

从左向右求解，结果不是0就是1，但表达式对象不但可以是0和1，还可以是0或非0的整数，也可以是字符型、浮点型、枚举型或指针型的纯量型数据

a && b && c：只有a为真时，才会判断b，a和b都为真时，才会判断c

a || b || c：若a为真，则不会判断b和c

表达式作逻辑值

表达式作逻辑值时，如if(a=1)，则 a=1 等价于 (a=1)!=0；!(a=1) 等价于 (a=1)==0；

条件运算符和条件表达式

表达式1 ? 表达式2 : 表达式3

例：(a>b)?a:b，如果a>b为真，则表达式的值为a，否则为b

  1. 条件运算符优先级高于赋值运算符，所以先求解，再赋值
  2. 条件运算符先求解表达式1，看真假情况，求解表达式2或3
  3. 可以不用把条件表达式赋给变量，可以在表达式2和3里直接给变量进行赋值

用switch语句实现

switch语句是多分支选择语句

  1. 执行顺序，先顺序查找各个 case ，如果没有匹配的，流程转去执行 default 后面的语句
  2. case 和 default 都是一个标号
  3. 可以没有 default 标号，当没有匹配的 case 常量时，则不执行任何语句
  4. 各个 case 标号出现次序不影响执行结果，可以先出先 default 标号
  5. 每一个 case 常量必须不相同
  6. case 标号只起标记的作用，不进行任何条件检查
  7. 如果各 case 子句中没有 break 语句，将向下连续输出
